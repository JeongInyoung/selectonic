selectonic

jQuery based plugin for making any list of items
selectable by mouse and keyboard.

Download production version (minified) or development version.
See docs and source on GitHub

Options

See all options in the docs on GitHub.

User can select many items in a list.
List loses focus when user clicks outside of the list.
List clears selection when user clicks outside of the list.
Possibility to use keys Up, Down and Home, End, PageUp, PageDown to move cursor (focus), select range of items by holding shift and select all items by pressing ctrl+a or cmd+a.
If keyboard: true and focused element is last in a list, pressing ↓ set focus to first element of the list; oppositely with ↑ key, when focus on a first element.
Matches child elemets, that will be served as clickable area inside each item to select/unselect these items.
mouseMode
 how toggle selection on an item
mouse click selects target item and clear other selection in a list; to add/remove item to selection user needs holds ctrl key (file manager style);
mouse click to select/unselect items (like group of checkboxes).
event
item will be selected when user push on left mouse button;
item will be selected when user click (press and release) on left mouse button;
unselected items will react on mousedown, but selected items will react on click (press and release). It can be usefull when there is another plugin, for example to dragging selected items.
keyboardMode
 how toggle selection on an item
pressing keys up/down selects target item and clear other selection in a list; to add/remove item to selection user needs holds shift key (file manager style);
keys ↑ ↓ and others only moves focus on items, to select/unselect items user needs press space;

Methods

See all methods in the docs on GitHub.

Detach handlers, clear data and remove plugin's classes

Clear selection in a list.

Get array of selected elements.

Get array of id's of selected elements.

Disable plugin, but keep data and save it state.

Turn on disabled plugin.

refresh

Refresh plugin data. That is necessary when some items in list was removed.

option

Get or set one ore more options.

Getter:

elem.selectonic("option", "mouseMode");

Setter:

    
    elem.selectonic("option", "mouseMode", "select");
    elem.selectonic("option", {
        loop: true,
        mouseMode: "select"
    });

cancel

Cancel last changes in a list or prevent it, if called in beforeSelect callback. Method can be call only inside callback functions:

    
    elem.selectonic({
        stop: function(event, ui) {
            if( condition ) {
                this.selectonic("cancel");
            }
        }
    });

Select method

You can select items by plugin interface. Plugin accept selector or element:

elem.selectonic(someElements);

elem.selectonic(":even");

elem.selectonic("li:eq(3)");
