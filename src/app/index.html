selectonic

jQuery based plugin for making any list of items
selectable by mouse and keyboard.

Download production version (minified) or development version.
See docs and source on GitHub

Begin

jQuery based plugin for making any list of items selectable by mouse and keyboard. It could be usefull in webapp where are different widgets like menus, dropdowns with keyboard input, lists with multiple selection and so on. It maybe too bold for just a simple menu in one place.

Examples

Selectonic help to inplement widgets with albility to select items. It could be usefull in webapp where are different widgets like menus, dropdowns with keyboard input, lists of items with actions and more. Lets see som examples!

Select

There are many implementations of select control, but almost all with the same flaw – lack of keyboard's support. See example.

It has full keyboard support: Arrows, PageUp, PageDown and HOME, END.
You could focus it by TAB, press arrow and choose item.

(function( $, window ) {
  'use strict';

  var keyCode = { UP:38, DOWN:40, ESCAPE:27, ENTER:13 },
  $document = $( window.document ),

  Select = function( elem ) {
    this.$el     = $( elem );
    this.button  = this.$el.find('.select-trigger');
    this.title   = this.button.find('.select-title');
    this.list    = this.$el.find('.select-group');
    this.isOpend = false;
    this.init();
  };


  Select.prototype.init = function() {
    var _this = this;
    
    this.button.on('blur', function( e ) {
      _this.close.call( _this, e );
    });

    this.$el
      .on('keydown', function( e ) {
        _this.keyHandler.call( _this, e );
      })
      .on('mousedown', function(e) {
        var isOnButton = _this.button[0] === e.target ||
          _this.title[0] === e.target;
        
        if ( isOnButton ) {
          if (!_this.isOpend) {
            e.stopPropagation();
            _this.open.call( _this );
          } else {
            _this.close.call( _this );
          }
          return;
        }
        e.preventDefault();
      })
      .on('mouseup', function(e) {
        e.stopPropagation();
      });
    
    $document.on('mouseup', function() { _this.close(); });
    
    // attach selectionc with options
    this.list.selectonic({
      multi: false,
      keyboard: true,
      keyboardMode: 'toggle',
      mouseMode: 'mouseup',
      focusOnHover: true,
      selectionBlur: false,
      focusBlur: true,
      
      // After plugin's initialisation
      create: function() {
        // we don't want for any list's 
        // reaction on keyboard in closed state
        this.selectonic('disable');
      },
      // When item of list has been selected
      select: function( e, ui ) {
        e.preventDefault();
        _this.selected = ui.target;
        _this.setValue( ui.items.html() );
        _this.close();
      },
      // After each works cycle
      stop: function( e ) {
        if ( e.which === keyCode.ENTER && _this.isOpend ) {
          _this.close();
        }
      }
    });
  };


  Select.prototype.keyHandler = function( e ) {
    var key = e.which;
    if ( !this.isOpend &&
      (key === keyCode.UP || key === keyCode.DOWN )
    ) {
      e.stopPropagation();
      e.preventDefault();
      this.open();
    }
    if ( this.isOpend && key === keyCode.ESCAPE ) {
      this.close();
      e.stopPropagation();
    }
  };


  Select.prototype.open = function() {
    this.$el.addClass('opend');
    // Enable selectionic on the list
    this.list.selectonic('enable');

    if ( this.selected ) {
      this.list
        // Set focus on selected element
        .selectonic('focus', this.selected)
        // and scroll to it
        .selectonic('scroll');
    }
    this.isOpend = true;
  };


  Select.prototype.close = function() {
    this.$el.removeClass('opend');
    // Disable plugin on the list
    this.list.selectonic('disable');
    this.isOpend = false;
  };


  Select.prototype.setValue = function( val ) {
    this.title.html( val );
  };


  $.fn.mySelect = function() {
    if (!this.length) { return this; }
    this.each(function() { new Select(this); });
    return $(this);
  };

})( jQuery, window );

Options

See all options in the docs on GitHub.

User can select many items in a list.
List loses focus when user clicks outside of the list.
List clears selection when user clicks outside of the list.
Possibility to use keys Up, Down and Home, End, PageUp, PageDown to move cursor (focus), select range of items by holding shift and select all items by pressing ctrl+a or cmd+a.
If keyboard: true and focused element is last in a list, pressing ↓ set focus to first element of the list; oppositely with ↑ key, when focus on a first element.
Matches child elemets, that will be served as clickable area inside each item to select/unselect these items.
mouseMode
 how toggle selection on an item
mouse click selects target item and clear other selection in a list; to add/remove item to selection user needs holds ctrl key (file manager style);
mouse click to select/unselect items (like group of checkboxes).
event
item will be selected when user push on left mouse button;
item will be selected when user click (press and release) on left mouse button;
unselected items will react on mousedown, but selected items will react on click (press and release). It can be usefull when there is another plugin, for example to dragging selected items.
keyboardMode
 how toggle selection on an item
pressing keys up/down selects target item and clear other selection in a list; to add/remove item to selection user needs holds shift key (file manager style);
keys ↑ ↓ and others only moves focus on items, to select/unselect items user needs press space;

Methods

See all methods in the docs on GitHub.

Detach handlers, clear data and remove plugin's classes

Clear selection in a list.

Get array of selected elements.

Get array of id's of selected elements.

Disable plugin, but keep data and save it state.

Turn on disabled plugin.

refresh

Refresh plugin data. That is necessary when some items in list was removed.

option

Get or set one ore more options.

Getter:

elem.selectonic("option", "mouseMode");

Setter:

    
    elem.selectonic("option", "mouseMode", "select");
    elem.selectonic("option", {
        loop: true,
        mouseMode: "select"
    });

cancel

Cancel last changes in a list or prevent it, if called in beforeSelect callback. Method can be call only inside callback functions:

    
    elem.selectonic({
        stop: function(event, ui) {
            if( condition ) {
                this.selectonic("cancel");
            }
        }
    });

Select method

You can select items by plugin interface. Plugin accept selector or element:

elem.selectonic(someElements);

elem.selectonic(":even");

elem.selectonic("li:eq(3)");
